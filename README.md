

# Singapore Taxi Availability
This webpage visualises available taxis in Singapore, updated every 30 seconds.

## System Architecture 
Resource List

* data.gov.sg API
* Leaflet
* Google Sheets as DB
* Jotform
* HTML/JS/CSS
* Public Cloud/Server
